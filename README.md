## About This Project

Patuakhali Science and Technology University is one of the fastest growing public universities in Bangladesh. The Faculty of Computer Science and Engineering (CSE) in this university offers a four-years graduation program. I am developing the software solution for this faculty which may have-

- [Students Enroll Management System]
- [Teachers Course Management]
- [Result Publishing System]
- [Attendnace System].
and so on. It is an ongoing project. We focus on optimized and Scalable Design.
